## OxiNest - Air Quality Monitor

OxiNest is a React Native app that monitors air quality across India using real-time data from WAQI (World Air Quality Index) API.

### Features:
- Real-time AQI data for 159+ Indian cities
- Interactive heatmap visualization
- Current location tracking
- Daily automatic data updates (3:00 AM)
- Clean, modern UI with theme support

### Data Sources:
- WAQI API for real-time air quality data
- Automatic daily updates via scheduled task
- Backup system with timestamped data files
